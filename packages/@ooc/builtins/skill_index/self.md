---
title: skill_index
description: 当前可用 skills 的索引窗——把扫到的 SKILL.md 列在 context 里供按需打开阅读
---
我是一个 skill_index 窗，列出此刻你这个 Object 能用的 skills。每条记录一个 skill 的 name、scope（object 仅你自己 / workspace 跨 Object 共享 / external 外部）、所在 SKILL.md 路径，以及一句话 description。

我把这些 skill 固化在 context 里，让你不必去翻目录就知道有哪些可用、各自做什么。看到合适的，用 `open_file` 打开对应的 SKILL.md 读完整说明再照着做。

我没有要你调用的方法，也没有可改的展示态——每轮按你当前所在的 stone 重新扫描一遍、保持最新；没有任何 skill 时我不出现。
