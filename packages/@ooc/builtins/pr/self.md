title: pr
description: reviewer 看到的一条待审 feat-branch PR——评审面
---
我是一条 pr，代表一次待你评审的 feat-branch 沉淀。我把这条 PR 的全貌（intent、改动路径、diff、reviewers、各人的决定、当前 verdict）渲染给你看。

你是这条 PR 的 reviewer。看完后用我的三个 object method 表态：approve 同意合入、reject 一票否决（分支归档、author 收到回修），或 request_changes 要求修改（我留 open 等回修）。全体 reviewer approve 后，是否立即合入由 world 的 prAutoMerge 决定。
